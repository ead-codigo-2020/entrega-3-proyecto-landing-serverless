Proveedores

Exsite una variedad de proovedores que ofrecen este tipo de funciones como servicio (FaaS).

A continuación encontrarás algunos de los más utilizados:

AWS Lambda

AWS Lambda es un servicio informático que permite ejecutar código sin aprovisionar ni administrar servidores. AWS Lambda ejecuta el código solo cuando es necesario, y se escala de manera automática, pasando de pocas solicitudes al día a miles por segundo. Solo se paga el tiempo de computación que se consume; no hay ningún cargo mientras el código no se ejecuta.

Sitio

Microsoft Azure

La plataforma Azure está compuesta por más de 200 productos y servicios en la nube diseñados para ayudarle a dar vida a nuevas soluciones que permitan resolver las dificultades actuales y crear el futuro. Cree, ejecute y administre aplicaciones en varias nubes, en el entorno local y en el perímetro, con las herramientas y los marcos que prefiera.

Sitio

Google Cloud Functions

Google Cloud Functions es la solución de procesamiento sin servidores de Google para crear aplicaciones controladas por eventos. Es un producto conjunto del equipo de Google Cloud Platform y el equipo de Firebase.

Sitio

Spotinst

Spotinst Elastigroup es un servicio de escalado de aplicaciones. Similar a los grupos de Auto Scaling de Amazon Elastic Compute Cloud (Amazon EC2), Spotinst Elastigroup está diseñado para optimizar el rendimiento y los costos, aprovechando instancias de Spot en combinación con instancias reservadas y bajo demanda.

Sitio

Apache OpenWhisk

Apache OpenWhisk es una plataforma sin servidor distribuida de código abierto que ejecuta funciones (fx) en respuesta a eventos a cualquier escala. OpenWhisk administra la infraestructura, los servidores y el escalado utilizando contenedores Docker para que pueda concentrarse en crear aplicaciones asombrosas y eficientes.

Sitio

Cloudflare

Cloudflare es una red global en la nube, inteligente e integrada, que proporciona seguridad, funcionamiento y fiabilidad para todas las aplicaciones de Internet, la infraestructura de red y los equipos.

Sitio

Firebase

Herramientas de Google cuya finalidad es desarrollar la infraestructura de aplicaciones, mejorar la calidad de estas y contribuir al crecimiento de un negocio

Sitio

knative

Los componentes Knative se construyen sobre Kubernetes, abstrayendo los detalles complejos y permitiendo a los desarrolladores centrarse en lo que importa. Construido mediante la codificación de las mejores prácticas compartidas por implementaciones exitosas del mundo real, Knative resuelve las partes "aburridas pero difíciles" de implementar y administrar servicios nativos en la nube para que usted no tenga que hacerlo.

Sitio

Kubeless

Kubeless es un marco sin servidor nativo de Kubernetes que le permite implementar pequeños fragmentos de código (funciones) sin tener que preocuparse por la infraestructura subyacente. Está diseñado para implementarse sobre un clúster de Kubernetes y aprovechar todas las excelentes primitivas de Kubernetes.

Sitio

Openfaas

OpenFaaS® simplifica la implementación de las funciones y el código existente en Kubernetes
